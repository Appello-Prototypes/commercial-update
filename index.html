Stalled / Effectively Dead — Enterprise Opportunities

These deals represent a combined ~$90-100k in potential ARR that has gone cold. Both share a nearly identical pattern: large companies (~200 workers), significant existing tech spend, genuine desire to streamline — but unwilling to absorb the integration cost-share.

Taurus Insulation (TX) Stalled ~$40-50k/yr ARR
~200 employees • Currently $138k/yr on NetSuite • Grew $2M → $18M revenue
  • What happened: Appello proposed subscription (~$45k/yr) + joint $40k cost-share ($20k each) for NetSuite integration
  • Taurus balked at the integration cost — combined with $17k onboarding, total first-year outlay was ~$80k+
  • Root cause: Wanted a turnkey solution at subscription cost, not a co-development partnership
  • Deal has stalled hard. No meaningful engagement. Should be moved to long-term nurture or closed-lost.
ProElectric (London, ON) Stalled ~$40-50k/yr ARR
~175 employees • Half HVAC / Half Electrical • Currently on Vista by Viewpoint
  • What happened: After two enthusiastic on-site meetings, Appello proposed subscription (~$50k/yr) + joint cost-share for Vista integration
  • Integration bill (~$40k total, 50/50 split) + $17k onboarding left a bad taste
  • Root cause: Same as Taurus — excited about platform, unwilling to pay for integration development
  • If Appello builds Vista integration independently, ProElectric could re-engage — but speculative
Pattern Analysis: Both Taurus and ProElectric expose a structural challenge in Appello's upmarket strategy. Companies with 200+ workers on enterprise ERPs (NetSuite, Vista) represent $40-50k ACV opportunities, but the integration cost-share model is not landing. These prospects want to see the integration already built before committing, not co-fund its development. This reinforces the ICP sweet spot of QBO-based companies where the native integration already exists.
